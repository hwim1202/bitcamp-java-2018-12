외부 라이브러리 사용 - 서버에 다운로드 받아 사용하기

- 외부 라이브러리를 서버에 다운로드 받는다
- 각 페이지의 라이브러리 경로는 서버에 다운로드 받은 경로이다. 

[장점]
- 외부 서버와 연결이 되어 있지 않아도 상관 없다.
- 외부에서 임의로 라이브러리 파일에 접근할 수 없다.
- 실무에서는 보안(은행, 관공서 등)과 안정성() 때문에 이 방식을 많이 사용한다
  
[단점]
- 라이브러리의 버전이 바뀔 때마다 직접 다운로드 해야 한다.
- 다른 사이트에서 같은 라이브러리를 다운로드 했더라도 우리 서버의 페이지에서 다시 다운로드 받아야 한다.
